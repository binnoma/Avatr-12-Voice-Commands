وصف المشروع
لوحة تحكم ويب (Web Dashboard) تفاعلية ومصممة بعناية لمالكي سيارة أفاتار 12 (Avatr 12). تساعدك هذه الأداة على التحكم في السيارة صوتياً باستخدام أوامر باللغة الصينية المتوافقة مع نظام Huawei HarmonyOS المدمج في السيارة.

✨ المميزات
🚀 يعمل بدون إنترنت (Offline-First): تم تصميم الكود بالكامل ليعمل بدون الحاجة لاتصال بالشبكة.
🌍 دعم ثنائي اللغة: عرض واضح للأوامر بالعربي والصيني لسهولة الفهم والاستخدام.
🎨 واجهة عصرية (Modern UI): تصميم يعتمد على أسلوب "الزجاج الشفاف" (Glassmorphism) مع ألوان مستقبلية تتناسب مع هوية السيارة.
🔊 نطق صوتي (Text-to-Speech): تقنية تحويل النص إلى كلام مدمجة، تنطق الأمر الصيني فور الضغط على الزر.
📱 متجاوب بالكامل (Responsive): يعمل بامتياز على شاشات الهواتف والشاشات الصغيرة أثناء القيادة.
🚀 كيفية الاستخدام
قم بتنزيل ملف index.html من هذا المستودع.
افتح الملف باستخدام أي متصفح ويب (مثل Chrome, Safari, Edge) على هاتفك أو جهاز الكمبيوتر.
اضغط على الأيقونة المطلوبة لسماع الأمر الصوتي.
تأكد من رفع صوت الجهاز لكي تلتقط السيارة الأمر عبر ميكروفونها.
🛠️ التقنيات المستخدمة
HTML5 (هيكلية المشروع)
CSS3 (التصميم المتجاوب والأنيميشن)
JavaScript ES6+ (التفاعل والنطق الصوتي)
📱 لقطات شاشة (للمستقبل)
يمكنك إضافة صور للواجهة هنا لاحقاً.
